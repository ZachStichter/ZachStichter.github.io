# Color Palette Reference - Option 1: Minimal Light

**Use case:** Neutral, timeless, minimal

## Colors
- **Base Background:** #ffffff
- **Primary Accent:** #40baec (cyan)
- **Card Background:** #f8f8f8 (light gray)
- **Borders/Dividers:** #e0e0e0 (medium gray)
- **Text (Dark):** #000000 (pure black)

## Notes
- Simplest palette
- Maximum contrast
- Pure white background
- Can switch to this anytime if monochromatic teal feels too much
