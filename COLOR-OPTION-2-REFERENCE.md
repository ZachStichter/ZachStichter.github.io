# Color Palette Reference - Option 2: Monochromatic Teal

**Use case:** Cohesive, sophisticated, cyan family. Currently implemented on site.

## Colors
- **Base Background:** #ffffff (white)
- **Primary Accent:** #40baec (bright cyan)
- **Card Background:** #d4eff9 (light teal)
- **Borders/Dividers:** #1a6f8a (deep teal, blue-heavy)
- **Text (Dark):** #0d3a42 (dark teal)

## Design Features
- Monochromatic teal family creates visual cohesion
- #40baec pops as primary accent without competing
- Light teal backgrounds (#d4eff9) provide subtle depth
- Dark teal text (#0d3a42) for sophisticated readability
- No box shadows (minimal, timeless look)
- Minimal border radius (2px on cards, 50% on circular elements)

## Notes
- Currently active on light mode
- Can pair with a complementary dark mode
- Timeless and sophisticated feel
- Teal/cyan trend has longevity compared to other colors
