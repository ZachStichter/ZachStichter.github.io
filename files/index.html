📌 How to add files

Copy one of the <a class="file-item"> examples below and update:

  • href — relative path to a local file (e.g. files/talk-2025.pptx) or an absolute URL.
  • id — a slug used to deep-link from other pages (e.g. id="presentation-slug").
  • Set download attribute on local file links so the browser downloads instead of trying to preview.
  • For online links (Google Slides, OneDrive, etc.) use target="_blank".

You can hide this block by adding style="display:none" to it.
